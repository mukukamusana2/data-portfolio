# Mukuka Musana — Data Analytics Portfolio
Welcome to my portfolio. Below are 4 data analytics projects showcasing my skills in SQL, Power BI, and data modelling.

## Olist Sales Performance Dashboard
Analysis of $13.2M in revenue across 96K orders on the Olist e-commerce platform. Built using PostgreSQL for data extraction and Power BI for visualisation, covering sales trends, top categories, and seller performance.

## Olist Customer Retention Analysis
Customer retention deep-dive using RFM segmentation and cohort analysis on the Olist platform. Identified key drivers of repeat purchases across delivery performance, review scores, and product categories.

##  Corporate Financial Statement Analysis (SEC EDGAR)
A 15-year comparative analysis of Costco, Target, and Walmart using SEC 10-K filings. Covers revenue growth, profit margins, return on equity, and debt ratios across three Power BI report pages.

## Top Economies in Asia
Macroeconomic analysis of 10 Asian economies using World Bank data. Explores GDP growth, FDI, inflation, unemployment, and export trends from the 1960s through to the post-COVID recovery.
